Application: ConversionWebApp
Simple WebApplication with HTML to show the conversions.
Purpose : to convert the temperature, distance and weight to their respective conversions
========================================================================================

Application Port 
--------------------

application start with 8080 port 

Apache Tomcat Server


to start the application 
--------------------------
run the Conversion.html on any server in local and access using the below URL

http://localhost:8080/ConversionWebApp/Conversion.html
